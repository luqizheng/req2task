# 前端测试方案

## 现状
- 前端项目无测试框架配置
- 无现有测试文件

## 推荐方案
使用 **Vitest**（单元测试）+ **Playwright**（E2E 测试）

## 实施步骤

### 1. 安装测试依赖
```bash
cd apps/web
pnpm add -D vitest @vue/test-utils jsdom @playwright/test
```

### 2. 创建 vitest.config.ts
- 配置 test environment 为 jsdom
- 配置 alias（@ 路径别名）
- 配置 coverage（可选）

### 3. 添加 package.json scripts
```json
{
  "test": "vitest",
  "test:run": "vitest run",
  "test:ui": "vitest --ui",
  "test:e2e": "playwright test"
}
```

### 4. 创建示例测试文件
- `src/components/__tests__/HelloWorld.spec.ts`

### 5. E2E 测试配置
- 创建 `playwright.config.ts`
- 配置 baseURL 和 webServer
- 示例测试 `e2e/example.spec.ts`

### 6. 运行命令
```bash
pnpm --filter @req2task/web test        # 单元测试
pnpm --filter @req2task/web test:e2e    # E2E 测试
```
