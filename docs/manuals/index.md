## 快速入门

### 首次使用流程

1. **创建项目** → 进入项目管理页面，创建新项目
2. **创建用户** → 进入用户管理页面，添加项目成员
3. **分配权限** → 在项目成员管理中设置角色
4. **开始工作** → 创建功能模块和需求
