# 项目管理

## 1. 创建项目

### 操作步骤

1. 进入 **项目管理** 页面
2. 点击 **创建项目** 按钮
3. 填写项目信息
4. 点击 **确定** 完成创建

### 必填字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 项目名称 | 项目的显示名称 | 电商平台后台系统 |
| 项目 Key | 唯一标识，大写字母开头 | PROJ / ECMS |

### 可选字段

| 字段 | 说明 |
|------|------|
| 描述 | 项目背景、目标等说明 |
| 状态 | 进行中 / 规划中 / 已完成 / 已归档 |

### 项目 Key 规则

- 格式：`[A-Z][A-Z0-9]*`
- 必须以大写字母开头
- 只能包含大写字母和数字
- 全系统唯一，不可重复

### 操作示例

```
项目名称：电商平台后台系统
项目 Key：ECMS
描述：支持商品管理、订单处理、用户运营的完整后台系统
状态：进行中
```

## 2. 查看项目列表

### 列表功能

- **搜索**：支持按项目名称或 Key 模糊搜索
- **筛选**：支持按状态筛选
- **分页**：支持自定义每页显示数量
- **排序**：按创建时间倒序排列

### 列表字段

| 字段 | 说明 |
|------|------|
| 项目名称 | 点击可进入项目详情 |
| 项目 Key | 项目的唯一标识 |
| 描述 | 项目简介（溢出显示...） |
| 状态 | 进行中/规划中/已完成/已归档 |
| 成员 | 头像组显示，最多显示3人 |
| 创建时间 | 项目创建日期 |
| 操作 | 查看详情 / 删除 |

## 3. 项目详情

### 详情页面结构

```
┌─────────────────────────────────────────┐
│  项目名称                        [返回] │
│                              [删除]     │
├─────────────────────────────────────────┤
│  [统计卡片] 需求数/任务数/完成率         │
├─────────────────────────────────────────┤
│  [快捷操作] 创建模块/创建需求/AI助手     │
├───────────────────────┬─────────────────┤
│  [项目信息]           │  [成员管理]     │
│  - 名称/Key           │  - 成员列表     │
│  - 描述               │  - 添加成员     │
│  - 状态               │  - 移除成员     │
├───────────────────────┴─────────────────┤
│  [功能模块]                              │
│  - 模块列表                              │
│  - 创建/编辑/删除模块                    │
└─────────────────────────────────────────┘
```

### 项目信息编辑

在项目详情页面可修改：
- 项目名称
- 项目描述
- 项目状态

## 4. 成员管理

### 添加成员

1. 进入项目详情页
2. 在 **成员管理** 卡片点击 **添加成员**
3. 在弹窗中选择用户
4. 点击 **添加** 完成

### 成员角色

| 角色 | 说明 |
|------|------|
| owner | 项目所有者，拥有最高权限 |
| manager | 项目管理员，管理项目成员和配置 |
| developer | 开发人员，参与需求和任务 |
| viewer | 访客，仅查看权限 |

### 移除成员

1. 在成员列表中找到目标成员
2. 点击 **移除** 按钮
3. 确认操作

---

## 5. 删除项目

### 注意事项

- 删除操作不可逆
- 会同时删除项目下的所有模块、需求、任务
- 建议在删除前确认项目内容已备份

### 操作步骤

1. 在项目列表或详情页点击 **删除**
2. 在确认弹窗中点击 **确定**
3. 项目及其所有关联数据将被删除
